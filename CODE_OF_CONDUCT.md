# Code of Conduct

## Our standard

This project adopts the **Contributor Covenant, version 2.1** as its code of conduct.
The full text is published here:

> https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: everyone participating in this project - issues, pull requests, discussions -
is expected to act with respect and good faith, keep the focus on the work, and help
make participation welcoming for people of all backgrounds. Disagreement is fine and
expected; treating people poorly is not.

## Scope

This applies in all project spaces (the repository, issues, pull requests, and any
related discussion) and when an individual is representing the project in public spaces.

## Reporting and enforcement

If you experience or witness unacceptable behavior, please report it privately to the
maintainer rather than in a public thread:

- Open a private report through the repository's **Security -> Report a vulnerability**
  tab, or
- Contact the maintainer through their GitHub profile: https://github.com/d0t0gg91-ux

Reports will be reviewed and handled as described in the Enforcement and Enforcement
Guidelines sections of the linked Contributor Covenant. The maintainer is responsible
for clarifying standards and may take appropriate, fair corrective action in response
to behavior that violates this code.
